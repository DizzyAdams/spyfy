# 📖 Glossário — SpyFy

Termos de domínio, técnicos e de negócio usados em toda a documentação.

## Domínio de Ads / Ofertas

| Termo | Definição |
|-------|-----------|
| **Ad (Anúncio)** | Uma peça publicitária individual (criativo + copy + destino). |
| **Creative (Criativo)** | O ativo visual do anúncio: imagem, vídeo, carrossel. |
| **Offer (Oferta)** | A proposta comercial completa: produto + preço + funil + copy. |
| **Angle (Ângulo)** | O ângulo persuasivo usado (ex.: "sem dieta", "em 21 dias"). |
| **Big Idea** | A grande ideia central que sustenta uma oferta. |
| **Hook (Gancho)** | Os primeiros segundos/frase que prendem a atenção. |
| **VSL** | Video Sales Letter — vídeo longo de vendas. |
| **TSL** | Text Sales Letter — carta de vendas em texto. |
| **Advertiser (Anunciante)** | A página/conta que veicula o anúncio. |
| **Longevity (Longevidade)** | Há quanto tempo o anúncio está ativo — sinal de sucesso. |
| **Winning Offer** | Oferta com alta longevidade e volume — comprovadamente lucrativa. |
| **Swipe File** | Coleção de referências salvas para inspiração. |
| **Nicho / Vertical** | Segmento de mercado (ex.: emagrecimento, finanças, relacionamento). |
| **Nutra** | Nicho de suplementos/nutracêuticos. |
| **Scaling (Escalando)** | Oferta aumentando volume de veiculação. |
| **Saturation (Saturação)** | Nível de concorrência/repetição de uma oferta no mercado. |

## Funil

| Termo | Definição |
|-------|-----------|
| **Landing Page (LP)** | Página de captura/venda de destino do anúncio. |
| **Upsell** | Oferta adicional após a compra. |
| **Downsell** | Oferta alternativa mais barata após recusa. |
| **Order Bump** | Adicional no checkout. |
| **Thank You Page (TY)** | Página de agradecimento pós-compra. |
| **Checkout** | Página/plataforma de pagamento. |
| **Funnel (Funil)** | Sequência completa: LP → checkout → upsell → downsell → TY. |

## Redes / Fontes

| Termo | Definição |
|-------|-----------|
| **Meta Ad Library** | Biblioteca pública de anúncios do Facebook/Instagram. |
| **TikTok Creative Center** | Repositório de anúncios do TikTok. |
| **Google Ads Transparency** | Central de transparência de anúncios do Google. |
| **Native Ads** | Anúncios em redes de conteúdo (Taboola, Outbrain). |

## Técnico

| Termo | Definição |
|-------|-----------|
| **Scraper** | Programa que extrai dados de páginas web. |
| **Crawler** | Programa que navega/descobre páginas automaticamente. |
| **Proxy Rotation** | Troca de IPs para evitar bloqueios. |
| **Headless Browser** | Navegador sem interface (Playwright/Puppeteer). |
| **Snapshot** | Captura versionada de um anúncio/LP em um momento. |
| **Enrichment** | Processo de enriquecer dados brutos (nicho, stack, idioma). |
| **Embedding** | Representação vetorial para busca semântica. |
| **Sub-Agent** | Agente de IA especializado numa tarefa (scrape, transcribe, clone). |
| **Fingerprint** | Assinatura técnica de uma LP (stack, pixels, fontes). |
| **Ingestion** | Entrada de dados brutos no pipeline. |
| **Idempotência** | Propriedade de operação que pode repetir sem efeito colateral. |
| **Backpressure** | Controle de fluxo quando consumidor é mais lento que produtor. |

## Negócio / Métricas

| Termo | Definição |
|-------|-----------|
| **WWOA** | Weekly Winning Offers Actioned — North Star Metric. |
| **TTFI** | Time-to-First-Insight. |
| **CPA** | Custo por aquisição. |
| **CPM** | Custo por mil impressões. |
| **MRR / ARR** | Receita recorrente mensal / anual. |
| **Churn** | Taxa de cancelamento. |
| **LTV** | Valor do tempo de vida do cliente. |
| **Activation** | Primeiro momento de valor entregue. |
